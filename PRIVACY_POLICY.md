# Privacy Policy

Last updated: April 21, 2026

## Overview

ChatGPT Anti-Lag does not collect, store, transmit, sell, or share personal data.

## What The Extension Does

ChatGPT Anti-Lag runs locally in your browser on `chatgpt.com` and `chat.openai.com`.
Its only purpose is to improve responsiveness in long ChatGPT conversations by hiding older messages and keeping fewer turns visible on the page.

## Data Collection

ChatGPT Anti-Lag does not collect personal information.
ChatGPT Anti-Lag does not collect chat content.
ChatGPT Anti-Lag does not collect browsing history outside the supported ChatGPT pages.
ChatGPT Anti-Lag does not use analytics, tracking, or advertising tools.

## Data Storage

The extension stores only your local preferences using Chrome extension storage, such as:

- whether Anti-Lag mode is enabled
- how many recent turns should remain visible

These settings stay in your browser storage associated with your Chrome profile and are not sent to any external server.

## Data Sharing

ChatGPT Anti-Lag does not sell data.
ChatGPT Anti-Lag does not transfer data to third parties.
ChatGPT Anti-Lag does not use data for advertising, profiling, or marketing.

## Remote Code And External Services

ChatGPT Anti-Lag does not use remote code.
ChatGPT Anti-Lag does not connect to external APIs or servers.

## Security

Because the extension does not transmit user data to external systems, there is no backend storage of your data by this extension.

If this privacy policy changes in a future release, the updated version will be included with the extension materials.
