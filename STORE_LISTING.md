# ChatGPT Anti-Lag Store Listing

## Name

ChatGPT Anti-Lag

## Short Description

Keeps long ChatGPT conversations responsive by hiding older messages and reducing on-page clutter.

## Full Description

ChatGPT Anti-Lag helps long conversations stay smooth.

When a ChatGPT thread gets very large, the page can start to feel heavy: typing slows down, scrolling becomes choppy, and the interface feels less responsive. This extension reduces that load by hiding older messages and keeping only the most recent turns visible.

What it does:

- hides older messages in long ChatGPT conversations
- lets you choose how many recent turns stay visible
- adds a small inline control inside the ChatGPT page
- works on `chatgpt.com` and `chat.openai.com`
- stores only your local settings with Chrome storage

What it does not do:

- does not read or send your chat data anywhere
- does not require an account beyond using ChatGPT itself
- does not connect to external servers

Best for:

- very long brainstorming sessions
- coding chats with lots of back-and-forth
- research threads that become slow over time

## Single Purpose

Improve ChatGPT performance in long conversations by hiding older messages and keeping the page lighter.
